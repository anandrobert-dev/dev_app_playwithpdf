# Oi360 PDF Suite

Premium PDF management application powered by GRACE.

## Features

- **PDF Splitter**: Split large PDFs into smaller files by page ranges
- **PDF Merger**: Combine multiple PDFs into a single document
- Drag and drop support
- Modern dark theme UI

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

## Requirements

- Python 3.8+
- PyQt5
- pypdf

## Folder Structure

```
playwithpdf/
```
